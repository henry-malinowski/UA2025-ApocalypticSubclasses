# UA2025-ApocalypticSubclasses

A Foundry implementation of the UA2025-ApocalypticSubclasses document released on D&amp;DBeyond by WoTC

### Subclasses

- Circle of Preservation (Druid)
- Gladiator (Fighter)
- Defiled Sorcery (Sorcerer)
- Sorcerer-King Patron (Warlock)

## Installation:

1. Switch to the Add-on Modules tab
2. Click Install Module
3. Paste the following URL into the Manifest URL textbox `https://github.com/henry-malinowski/UA2025-ApocalypticSubclasses/releases/latest/download/module.json`
4. Click Install
5. In a DND5e Foundry world enable the "UA2025: Apocalyptic Subclasses" module
